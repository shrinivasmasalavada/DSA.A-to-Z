# DSA (Data Structures & Algorithms) — A to Z Roadmap

## What is DSA?

DSA stands for:

* **Data Structures** → How data is stored
* **Algorithms** → Step-by-step problem-solving methods

DSA is important for:

* Software Engineering
* AI Systems
* Embedded Systems
* Competitive Programming
* Interviews 
* Robotics
* Backend Development

# 2. Time Complexity

Very important topic.

Learn:

* Big O Notation
* Time Complexity
* Space Complexity

Examples:

| Complexity | Speed     |
| ---------- | --------- |
| O(1)       | Fastest   |
| O(log n)   | Very Fast |
| O(n)       | Good      |
| O(n²)      | Slow      |
| O(2ⁿ)      | Very Slow |
